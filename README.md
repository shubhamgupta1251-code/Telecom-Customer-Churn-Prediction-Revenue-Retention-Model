# 📊 Corporate Customer Churn & Retention Analytics Framework

### 🎯 Business Objective
This project builds a predictive statistical model to identify high-risk subscription clients before they cancel service. This allows marketing teams to deploy proactive retention discounts, directly safeguarding monthly recurring revenue.

### 🔬 Statistical Insights & Output Verified
* **Overall Corporate Baseline Churn Rate:** 26.54% (Roughly 1 in 4 customers leave the firm).
* **Contract Risk Matrix:** Month-to-month users exhibit a high **42.7% churn risk**, whereas two-year contract signees drop to a stable **2.8% churn rate**.
* **Model Predictive Power:** Achieved an **ROC-AUC Score of 0.8328**, indicating highly reliable customer classification power.
* **Key Statistical Drivers:** Increased tenure mathematically decreases churn, while higher monthly charges significantly accelerate customer attrition.

### 💻 Technical Toolkit Applied
* **Language:** Python 3
* **Core Packages:** Pandas (Data Imputation), Scikit-Learn (Logistic Regression Optimization), NumPy, Seaborn.
* **Development Workspace:** Google Colab Cloud Environment
